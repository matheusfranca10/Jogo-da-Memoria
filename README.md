# Jogo da Memória

## Sobre o Projeto

Este é um **Jogo da Memória** desenvolvido com **HTML**, **CSS** e **JavaScript**. O objetivo do jogo é encontrar pares de cartas idênticas, desafiando a memória do jogador. A cada rodada, as cartas são viradas aleatoriamente e o jogador precisa encontrar os pares correspondentes.

## Funcionalidades

- **Cartas Aleatórias:** As cartas são viradas aleatoriamente a cada jogo.
- **Jogo Interativo:** O jogador deve encontrar os pares correspondentes de cartas.
- **Interface Simples:** Interface limpa e fácil de usar.

## Tecnologias Utilizadas

- **HTML5:** Estruturação do conteúdo.
- **CSS3:** Estilização e layout.
- **JavaScript:** Lógica de interatividade do jogo.
